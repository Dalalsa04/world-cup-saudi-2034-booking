# World Cup 2034 Match Booking Platform

## Project Description
This project is a web application designed to help users book tickets for FIFA World Cup 2034 matches in Saudi Arabia.

## Problem
Users may get confused when booking tickets due to unclear match information such as stadium, date, and teams. This can lead to booking the wrong match.

## Solution
This platform provides a clear and user-friendly interface that shows match details (teams, date, stadium) and helps users book tickets easily without confusion or mistakes.

It also includes helpful features such as organizing matches by city, a personal schedule to avoid time conflicts, and match reminders, which improve the overall booking experience.

## Target Users
This website is for people in Saudi Arabia, tourists, and football fans who want to book World Cup 2034 match tickets easily.

## Features
- Easy ticket booking  
  Simple and fast booking process without confusion.

- Clear match details  
  Shows teams, date, and stadium clearly to avoid mistakes.

- View all match dates  
  Users can see all matches and choose suitable times.

- Personal schedule  
  Allows users to organize matches and avoid time conflicts.

- Matches by city  
  Matches are divided by city to help users plan their trip easily.

- Match reminders  
  Users get reminders before the match time.
